About Me

A former middle school teacher who transitioned into tech two years ago, I am hooked on problem-solving and puzzle-working, finding bugs and improving functionality, and explaining integration possibilities to others.

On a usual day in the workplace, you can find me doing one of the following:

  • Integrating multiple endpoints or SQL queries together into one usable endpoint using vanilla JavaScript
  • Building custom React and JQuery apps
  • Talking to potential customers about both the big-picture and precise details about my company's product
  • High-fiving coworkers (virtually) over newly debugged code and/or the latest awesome talk with potential clients

Work samples

Weekend Adventure Hub

React, Redux, SCSS, API calls

I designed and built the entire front-end of this app using React, drawing from three different APIs to highlight outdoorsy activities along Colorado's Front Range. Behind the scenes, Redux organizes API calls and maintains state in a single store.

Blue Sky Catastrophe

React, SCSS

I built this website for a photographer friend with the intent to pass the repository to him once I was done building out and styling the components. Why React instead of HTML? Reusability and sustainability. The use of components made it easier both to update content and to transition between content managers.

Snowboard App

Node, React, and SCSS

I planned and built this fullstack app not only as a homage to powder days, but also to show off CRUD operations and authentication on both the front and back end.

Key Conservation Mobile

Node.js, React-Native, Redux

This is an app used by conservationist organizations to connect with potential supporters and donors. I made all necessary changes to the database during my two-month tenure and rigged up the navigation for maps and onboarding.
